# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in RipperMod Manager, please report it through
[GitHub Security Advisories](https://github.com/y4rd13/rippermod-manager/security/advisories/new).

**Please do not open a public issue for security vulnerabilities.**

We will acknowledge your report within 48 hours and provide a detailed response within 7 days,
including next steps and an expected timeline for a fix.

## Scope

- Backend API (FastAPI / Python)
- Frontend (React / TypeScript)
- Desktop shell (Tauri / Rust)
- Stored credentials and API keys
